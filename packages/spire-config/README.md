## Usage

1. Install both `@researchgate/spire-config` and `spire` as dev dependencies:

```sh
yarn add --dev @researchgate/spire-config spire
```

2. Set `spire` config in `package.json`

```json
{
  "spire": {
    "extends": "@researchgate/spire-config"
  }
}
```
